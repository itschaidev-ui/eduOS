# Quick Installation Guide

## Install the Extension

1. **Open Chrome Extensions Page**
   - Go to `chrome://extensions/`
   - Or: Chrome Menu → Extensions → Manage Extensions

2. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top right corner

3. **Load the Extension**
   - Click "Load unpacked"
   - Navigate to and select the `chaimode-extension` folder
   - Click "Select Folder"

4. **Done!**
   - The extension icon should appear in your Chrome toolbar
   - Click it to access all ChaiDev projects

## Keyboard Shortcuts

After installation, you can use:
- **Ctrl+Shift+E** (Mac: Cmd+Shift+E) - Open eduOS
- **Ctrl+Shift+L** (Mac: Cmd+Shift+L) - Open Lunchbox AI  
- **Ctrl+Shift+C** (Mac: Cmd+Shift+C) - Open chaimode

## Features

- Click any project card to open it instantly
- "Open All" button launches all projects at once
- Clean, dark-themed UI matching ChaiDev branding
- No need to keep websites open - access everything from the extension

## Troubleshooting

**Extension not showing?**
- Make sure Developer Mode is enabled
- Check that you selected the correct folder (should contain manifest.json)

**Icons missing?**
- The extension works without custom icons
- Chrome will use default icons
- To add custom icons, place PNG files in the `icons/` folder

**Keyboard shortcuts not working?**
- Go to `chrome://extensions/shortcuts`
- Find "ChaiDev Quick Access" and configure shortcuts
