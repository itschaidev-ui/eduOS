# Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can create simple icons using:
1. Online tools like [Favicon Generator](https://favicon.io/)
2. Design tools (Figma, Canva)
3. Or use emoji/unicode characters as placeholders

For now, the extension will work without icons (Chrome will use a default).
